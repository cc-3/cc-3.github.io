# Directorio de Instalación

El objetivo de este tutorial es dejar preparado el material que necesitarán para los laboratorios y proyectos.

Busca tu arquitectura en la tabla para acceder a la guía de instalación correspondiente,

| OS | Arquitectura | Recomendación | Guía de Instalación | Comentarios |
| :--- | :--- | :--- | :--- | :--- |
| **Windows** | Intel/AMD | Maquina virtual | [Máquina virtual Intel/AMD](intel-amd-vm.md) | Esta es la opción preferida.|
| **Windows** | ARM | No hay soporte oficial | | Comuníquese con su catedrático para instrucciones. |
| **Linux** | Intel/AMD | Instalación nativa | [Nativo para Linux](linux-native.md) | Soporte para Ubuntu 24. Otras distros son responsabilidad del usuario. |
| **Linux** | ARM | No hay soporte oficial | | Comuníquese con su catedrático para instrucciones. |
| **macOS** | Intel/AMD | Maquina virtual | [Máquina virtual Intel/AMD](intel-amd-vm.md) | Utilizará la misma máquina virtual que usuarios de Windows.  |
| **macOS** | ARM (Apple Silicon) | Maquina virtual | [Máquina virtual ARM](arm-vm.md) | Autograder local no funciona para labs de RISC-V. |
